# Chimera AI — Telegram-бот для прогнозирования ставок

**Chimera AI** — профессиональный Telegram-бот для анализа спортивных матчей и поиска ставок с положительным EV. Архитектура «Три Головы»: три независимых AI-агента совместно анализируют матчи и выдают сигналы.

## Поддерживаемые виды спорта

| Спорт | Лиги |
|-------|------|
| ⚽ Футбол | АПЛ, Ла Лига, Бундеслига, Серия А, Лига 1 и др. |
| 🏒 Хоккей | NHL, SHL, AHL, Finnish Liiga, Allsvenskan |
| 🏀 Баскетбол | NBA, Евролига, NBL Australia |
| 🎾 Теннис | Все активные ATP/WTA турниры |
| 🎮 CS2 | Все крупные турниры (PandaScore + HLTV) |

## Архитектура

### Ансамбль моделей (футбол)

| Компонент | Вес | Описание |
|-----------|-----|---------|
| Пуассон + xG | 40% | xG данные Understat, поправка Dixon-Coles |
| ELO + Форма | 30% | Динамические рейтинги, бонус формы последних 5 матчей |
| AI-агенты | 15% | GPT-4 (Химера) + Llama 3.3 70B (Тень) |
| Букмекеры | 10% | No-Vig вероятности, рыночная мудрость |
| Нейросеть | 5% | Prophet LSTM, 66k матчей АПЛ |

### AI-агенты («Три Головы»)

- **Статистик** — математический анализ: ELO, xG, форма команды
- **Скаут** — качественный анализ: новости, травмы, мотивация
- **Арбитр** — синтез + Kelly criterion + финальный вердикт
- **Тень (Llama 3.3 70B)** — полностью независимое второе мнение через Groq

### Сигнальная система

- **Футбол**: минимум 4 из 6 сигналов для рекомендации ставки
- **CS2**: минимум 6 из 10 сигналов (строже)
- **Порог ценности**: EV > 10%, коэф ≥ 1.55, уверенность ≥ 52%
- **Контрарная ставка**: при завышенном кэфе на андердога — отдельное предупреждение

## Установка

### 1. Клонировать репозиторий
```bash
git clone https://github.com/gitkit11/botV02.git
cd botV02
git checkout chimera-v2
```

### 2. Создать .env файл
```bash
cp .env.example .env
# Заполнить своими ключами
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
npm install
```

### 4. Запустить
```bash
python main.py
```

## Переменные окружения

| Переменная | Сервис |
|-----------|--------|
| `TELEGRAM_TOKEN` | Telegram Bot API |
| `THE_ODDS_API_KEY` | The Odds API (матчи + коэффициенты) |
| `OPENAI_API_KEY` | OpenAI GPT-4 (Химера) |
| `GROQ_API_KEY` | Groq Llama 3.3 70B (Тень) |
| `API_FOOTBALL_KEY` | API-Football (статистика) |
| `RAPID_API_KEY` | RapidAPI |
| `PANDASCORE_API_KEY` | PandaScore (CS2) |

## Команды бота

| Команда | Описание |
|---------|---------|
| `/start` | Главное меню |
| `/football` | Анализ футбольных матчей |
| `/hockey` | Анализ хоккейных матчей |
| `/basketball` | Анализ баскетбольных матчей |
| `/tennis` | Анализ теннисных матчей |
| `/cs2` | Анализ CS2 матчей |
| `/stats` | Статистика прогнозов: точность, ROI |
| `/signal` | CHIMERA Score — сигнал дня |
| `/express` | Построитель экспрессов |

## Автоматические задачи

- **Каждый час** — проверка результатов, обновление ROI в БД
- **Ежедневно** — синхронизация статистики HLTV (CS2)
- **Понедельник 3:00** — перекалибровка ELO

```
  hockey/                — модуль хоккей

chimera_signal.py        — CHIMERA Score engine

ml/                      — ML модели XGBoost/нейросети
```
