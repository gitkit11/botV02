'''
# Chimera AI - Telegram-бот для прогнозирования ставок

**Chimera AI** — это продвинутый Telegram-бот для анализа и прогнозирования футбольных матчей. Он использует **тройную систему искусственного интеллекта** для предоставления комплексных и глубоких прогнозов, выходя за рамки простой статистики.

## Архитектура "Тройного ИИ"

Проект состоит из трех независимых, но взаимосвязанных модулей ИИ:

1.  **🔮 ИИ #1 "Пророк"**:
    *   **Технология**: Нейросеть на базе **TensorFlow (LSTM)**.
    *   **Задача**: Анализирует более 30 лет исторических данных (66,000+ матчей), чтобы выявить скрытые закономерности, тренды и долгосрочную форму команд.

2.  **📰 ИИ #2 "Оракул"**:
    *   **Технология**: Модель обработки естественного языка **DistilBERT (Transformers)** и **GNews API**.
    *   **Задача**: Сканирует новостные ленты и статьи в реальном времени, чтобы оценить "моральный дух" команды. Анализирует тональность новостей (позитивная/негативная), выявляя информацию о травмах, конфликтах в команде или, наоборот, о высоком боевом духе.

3.  **⚖️ ИИ #3 "Маэстро"**:
    *   **Технология**: Ансамблевая модель с весовыми коэффициентами и алгоритм поиска **Value Bets**.
    *   **Задача**: Выступает в роли финального арбитра. Он объединяет числовые прогнозы "Пророка" и качественный анализ "Оракула", присваивая им веса. Затем он сравнивает полученную вероятность с коэффициентами букмекеров, чтобы найти **Value Bets** — ставки, где, по мнению ИИ, букмекер недооценил вероятность исхода.

## Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone <URL репозитория>
cd <название папки>
```

### 2. Установите зависимости

Убедитесь, что у вас установлен Python 3.10+.

```bash
pip install -r requirements.txt
```

### 3. Настройте API-ключи

Переименуйте `config.py.example` в `config.py` и вставьте ваши API-ключи:

```python
# config.py
TELEGRAM_TOKEN = "ВАШ_ТЕЛЕГРАМ_ТОКЕН"
THE_ODDS_API_KEY = "ВАШ_КЛЮЧ_THE_ODDS_API"
RAPID_API_KEY = "ВАШ_КЛЮЧ_RAPIDAPI"
```

### 4. Обучите ИИ #1 "Пророк"

Этот шаг нужно выполнить только один раз. Скрипт автоматически скачает датасет с Kaggle и обучит модель.

```bash
python train_prophet.py
```

После выполнения в папке появится файл `prophet_model.keras`.

### 5. Запустите бота

```bash
python main.py
```

Бот будет запущен и готов к работе в Telegram.
